# Retail Industry Sales & Cohort Analysis

## Project Overview
This project focuses on analyzing retail sales data to uncover trends, customer behavior, and business insights. The analysis supports data-driven decision-making by identifying key performance metrics and patterns.

## Tools & Technologies
- SQL (data querying and transformation)
- Microsoft Excel (data cleaning and validation)
- Power BI (interactive dashboards and visualizations)

## Key Analysis Performed
- Sales trend analysis (daily, monthly, and category-level)
- Customer cohort analysis to track retention and repeat purchases
- Revenue and order volume KPIs
- Data cleaning and preprocessing using SQL queries

## Dashboard
The Power BI dashboard includes:
- Total revenue, orders, and average order value
- Category-wise and region-wise sales performance
- Customer cohort retention matrix
- Time-based sales trends

## Files in This Repository
- `SQLQuery_data_cleaning_of_Retail_Industry.sql` – SQL queries for data cleaning and analysis
- Dashboard screenshots – Sample visuals from the Power BI dashboard

## Key Insights
- Identified high-performing product categories contributing majority of revenue
- Observed customer retention patterns across cohorts
- Highlighted seasonal sales spikes useful for inventory planning
